# AI Styler

Streamlit 기반의 한국 스타일 의류 추천 애플리케이션입니다. 사용자는 보유 중인 옷장을 업로드하고 자동 분류, 보유 의류로 만들 수 있는 코디 추천, 무신사·크림 스타일을 참고한 신규 아이템 추천을 한 곳에서 확인할 수 있습니다.

## 주요 기능
- **나의 옷장**: 이미지 업로드와 메타데이터 입력을 통해 의류를 등록하고 카테고리·스타일 태그를 확인합니다.
- **보유 코디 추천**: 저장된 아이템을 조합하여 색상·스타일·계절 정보를 바탕으로 코디를 제안합니다.
- **추천 아이템**: 부족한 카테고리를 분석하고 무신사 분류 체계를 따른 신규 아이템과 구매 링크를 제공합니다.

## 폴더 구조
```
AI styler/
├── assets/            # 업로드 이미지 및 샘플 에셋
├── data/              # 인벤토리 DB 및 추천용 카탈로그
├── models/            # 스타일 태깅 및 추천 모델 파일(추후 확장)
├── src/
│   ├── app.py         # Streamlit 진입점
│   └── services/      # 인벤토리/추천/마켓플레이스 로직
└── requirements.txt   # 의존성 목록
```

## 실행 방법
```bash
pip install -r requirements.txt
streamlit run src/app.py
```
